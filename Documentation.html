SeumX HTMLPad

This software was developed to help users create documents using Htlm tools. All the needed tools are available on the interface and you can create and style up your document. This program was developed to save files in Html format(.html)  and Source Code(.txt). 

The Copy,Cut,Paste,Undo,Redo and Delete Functions.

These six functions are currently disabled in this version of SeumXPad. We are working to implement more updates but currently users are advised to use the System key combination to perform these actions or use interface copy,cut and paste on functions. Following is a sammary of System shortcuts.

1. Copy : Press CTRL+C to copy a selected text.

2. Cut : Press CTRL+X to cut a selected text.

3. Paste : Press CTRL+V to paste a copied OR cut text.

4. Undo : Press CTRL+Z to undo a recent operation.

5. Redo : Press CTRL+Y to redo your undo operaton.

6. Delete : Press the Delete Key to delete a selected text.
